Projekt rozwiązywacza gry typu boggle, znajdujący wszystkie możliwe odpowiedzi.
Planszę możesz edytować w pliku "input.txt", albo dowolnym innym(program zapyta Cię o nazwę pliku) zachowującym formatowanie:

a b c d
e f g h
i j k l
m n o p

W pliku o wybranej przez Ciebie nazwie znajdziesz wszystkie możliwe słowa, które można zbudować z liter na planszy.
Z tego co wiem jest to jedyny taki projekt dla gry w języku polskim.


Project of a boggle-type game solver, finding all the possible answers.
You can edit the board in the "input.txt" file, or any other (the program will ask you for the file name) keeping the formatting:

a b c d
e f g h
i j k l
m n o p

In the file with the name of your chosing, you will find all the possible words that can be built from the letters on the board.
As far as I know, this is the only such project for the game in the Polish language.
